# System Overview

Arch Linux on Wayland with Hyprland as the compositor/window manager. Shell is bash, editor is Neovim (kickstart.nvim), terminal is ghostty.

## Desktop Tools

- Launcher: wofi
- Status bar: waybar
- Notifications: mako
- Wallpaper: hyprpaper
- File manager: Thunar
- Browser: brave
